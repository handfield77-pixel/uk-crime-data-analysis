# West Yorkshire Crime Data Analysis

## Overview
This project analyses west yorkshire crime data using R to identify crime trends, patterns, and hotspots.

## Tools Used
- R
- RStudio
- ggplot2
- dplyr

## Skills Demonstrated
- Data Cleaning
- Data Analysis
- Data Visualisation
- Statistical Reporting

## Project Files
- Analysis scripts
- Visualisations
- Final report

## Key Findings
This project explores crime trends across different locations and time periods to provide meaningful insights from real-world data.
